A data analysis project exploring Indian Premier League (IPL) data to uncover player performances, team trends, and match-winning factors using Python.
Project overview:
It aims to answer key questions such as: Which teams have the highest winning percentage?, How does winning the toss affect the match outcome?, Venue-based performance analysis.
Tech Stack used:
Language: Python
Libraries: pandas, numpy, matplotlib.
Environment: Jupyter Notebook
Key Insights-
Toss Impact: Analyzed whether "Toss Winner, Match Winner" is a myth or reality.
Top Performers: Identified top run-scorers and leading wicket-takers across seasons.
Venue Analysis: Determined which stadiums are "batting paradises" vs. "bowlers' graveyards".
